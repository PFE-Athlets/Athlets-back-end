# Athlets Backend
A basic, lightweight Spring Boot backend service. This basic backend was generated by springboot.

## 🚀 Tech Stack
* **Java:** 21
* **Framework:** Spring Boot
* **Build Tool:** Maven

## 📋 Prerequisites
Ensure you have the following installed on your local machine:
* [Java 21 SDK](https://adoptium.net/)
* Maven

## 🛠️ Getting Started

### 1. Clone the repository

Clone the repo and then head into it.
```bash
git clone <repository-url>
cd <project-directory>
```

### 2. Run the application locally

Run the springboot project, which will start springboot on port 8080.
```bash
mvn spring-boot:run
```
